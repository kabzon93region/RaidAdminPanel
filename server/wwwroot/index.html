Raid Admin Panel

Админка сервера SPT 4 + Fika — профили, рейды, принудительные действия

Быстрые действия

Сохранение профиля пишет текущее состояние из памяти сервера на диск. Extract и снимок инвентаря требуют клиентский мод RaidAdminPanel у игрока в рейде.

Активные рейды (Fika)

«Высадка всем» ставит команду на клиент: живые игроки завершают рейд как survived (как F8/extract в Fika). Мёртвые клиенты команду не выполнят.
